# Agent Economy blog focus

Use this as the editorial baseline when curating Hacker News.

## Core themes

- AI infrastructure
- security
- trust
- commercialization
- applications
- trading / transactions

## Good fits

- Frontier model launches with concrete product or infrastructure implications
- Agent tooling, orchestration, protocols, memory, evals, deployment
- Security architecture, identity, verification, privacy, provenance
- Developer tools that materially change software production
- Market structure shifts: cloud, chips, APIs, pricing, monetization, distribution
- Strong essays on how AI changes work, business models, or coordination

## Weak fits

- Consumer internet fluff with no strategic angle
- Generic startup advice
- Pure entertainment, gossip, outrage, or meme threads
- Hobby demos with little broader significance

## Preferred output style

Curate for signal.
Do not try to be comprehensive.
Pick the few items Terry could realistically build a worthwhile post around.
