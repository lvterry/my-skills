# My Skills

Personal OpenClaw skills for Terry.

## Skills

- **blog-publisher** - 自动添加文章到 Agent Economy 博客
- **course-learning** - 跟踪课程学习进度
- **notes** - 保存和管理个人笔记
- **notion-clip** - 保存推文和笔记到 Notion
- **portfolio-tracker** - 查询股票持仓市值
- **spawn-agent** - 自动派生子代理
- **twitter-archiver** - 提取和归档 Twitter 内容
- **video-archiver** - 下载视频并上传到 Google Drive

## Usage

这些技能位于 `~/.openclaw/skills/` 目录，OpenClaw 会自动加载。
